# LiteObsidian 后续开发任务文档

## 说明
这个文档只放本轮没有指定实现的功能，作为后续待办池。

## 待办功能池
1. 系统分享接入：从其他应用分享文本和链接到新建笔记
2. 深链能力：支持通过链接直接打开指定笔记
3. 应用安全：应用锁、指纹或人脸解锁
4. 快捷入口：桌面快捷方式和小组件
5. 历史快照：笔记版本历史和回滚
6. 批量管理增强：批量移动、批量加标签、批量归档
7. 搜索增强：高亮、筛选组合、全文检索优化
8. 预览增强：目录导航、编辑预览同步滚动
9. 主题扩展：自定义主题和配色方案
10. 附件管理增强：附件浏览器、空间占用统计、清理工具

## 排期建议
1. 优先级 P1
系统分享接入、深链能力、搜索增强
2. 优先级 P2
应用安全、快捷入口、预览增强
3. 优先级 P3
历史快照、主题扩展、附件管理增强
